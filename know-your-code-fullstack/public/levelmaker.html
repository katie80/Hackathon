Which function returns the range of the following array? The range is the difference between the smallest and largest number. findRange([3, 7, 3, 9, 12, 14, 2, 10]); The range of this array is 12, (14 - 2). 

function findRange(arr) {
var min = arr[0];
var max = arr[0];
for(var a of arr) {
if(a > max) {
max = a;
} else if (a < min) {
min = a;
}
}
return max - min;
}
